HTML (HyperText Markup Language)

HTML - это язык разметки, который используется для создания структуры веб-страниц. Основные элементы включают:

  • <html> - Корневой элемент, содержащий весь HTML-документ.
  • <head> - Секция для мета-информации, скриптов и стилей.
  • <title> - Заголовок страницы, отображаемый в браузере.
  • <body> - Основное содержимое страницы.
  • <h1> - <h6> - Заголовки от самого большого (<h1>) до самого маленького (<h6>).
  • <p> - Абзац текста.
  • <a> - Ссылка на другую страницу или ресурс.
  • <img> - Изображение на странице.
  • <ul>, <ol>, <li> - Неупорядоченный и упорядоченный списки.
  • <div>, <span> - Блочные и строчные контейнеры для элементов.
  • <footer> - Элемент футера, который обычно содержит контактную информацию, авторские права и другие сведения, располагаемые внизу страницы.

Пример HTML-документа

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Пример страницы</title>
</head>
<body>
    <h1>Заголовок страницы</h1>
    <p>Это пример абзаца текста.</p>
    <a href="https://example.com">Ссылка на сайт</a>
    <ul>
        <li>Элемент списка 1</li>
        <li>Элемент списка 2</li>
    </ul>
    <footer>
        <p>© loga4</p>
    </footer>
</body>
</html>

CSS (Cascading Style Sheets)

CSS используется для стилизации HTML-документов. С его помощью можно изменять внешний вид элементов, включая цвета, шрифты, отступы и расположение на странице.

Пример подключения CSS

Стили можно подключить различными способами:

  • Встроенные стили - использование атрибута style внутри HTML-тега.
  • Внутренние стили - определение стилей внутри тега <style> в HTML-документе.
  • Внешний файл стилей - подключение внешнего файла с расширением .css.

Пример использования встроенных стилей:

<p style="color: blue; font-size: 18px;">Этот текст синим цветом и размером шрифта 18px.</p>

Этот текст синим цветом и размером шрифта 18px.

Пример использования внутренних стилей:

<style>
    p {
        color: blue;
        font-size: 18px;
    }
</style>
<p>Этот текст синим цветом и размером шрифта 18px.</p>

Этот текст синим цветом и размером шрифта 18px.

Пример подключения внешнего файла стилей:

<link rel="stylesheet" href="styles.css">

CSS-селекторы

Селекторы используются для указания, к каким элементам HTML следует применять стили:

  • Тег - селектор по тегу, например, p.
  • Класс - селектор по классу, например, .classname.
  • ID - селектор по ID, например, #idname.

Пример использования CSS-селекторов:

<style>
    p {
        color: red;
    }
    .highlight {
        background-color: yellow;
    }
    #unique {
        font-weight: bold;
    }
</style>

<p>Обычный текст</p>
<p class="highlight">Выделенный текст</p>
<p id="unique">Уникальный текст</p>

Основные свойства CSS

  • color - цвет текста.
  • background-color - цвет фона.
  • font-size - размер шрифта.
  • margin - внешние отступы элемента.
  • padding - внутренние отступы элемента.
  • border - граница вокруг элемента.
  • display - тип отображения элемента (например, block, inline, none).

Пример использования CSS-свойств:

<style>
    .box {
        color: white;
        background-color: blue;
        font-size: 20px;
        padding: 10px;
        margin: 20px;
        border: 2px solid black;
        display: inline-block;
    }
</style>

<div class="box">Это стилизованный блок</div>
